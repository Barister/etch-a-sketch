Etch A Sketch

Welcome to my home project from The Odin Project Foundations curriculum, Etch-A-Sketch like app.

You can draw in two different modes: using the mouse (Mouse button) or the wheels below the screen (Wheels button).

In Mouse mode, right-click erases the color of one cell.

For drawing in Wheels mode, use the A and Z buttons for horizontal drawing and the K and M buttons for vertical drawing. You can also try controlling the wheels with the mouse.

A cell can be filled with black color (Black button) or a random color from the RGB palette (Color button).

If you color the grid again, it will become darker towards black by about 10%.

To clear the screen, use the top slider. Slide it all the way to the right and back to start draw again.

The right slider changes the screen resolution.
